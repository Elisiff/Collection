Страница

<html> задаем класс .page и на него записываем глобальные стили страницы.

  • Нормализация вьюпорта для windows-телефонов.
  • box-sizing: border-box;
  • Фиксированный скроллбар.
  • Убираем подсветку ссылок и JS-элементов в WebKit на мобильных платформах.
  • Задаем правильное масштабирование текста для IE на мобильных устройствах.
  • Сброс отступов.
  • Стили для <img>.
  • Запрет масштабирования по двойному клику на сенсорных экранах.
  • Стили для IE.

Текстовые блоки

Посты

.post

С помощью flexbox можно регулировать высоту блоков с текстом и расстояние между отдельными фрагментами внутри.

Вчера он на луну летал, сегодня в руки к нам попал.

Соседи будут рады!

Фигуры на CSS

Бургер

.burger

Бургер (кнопка) состоит из одной полоски (псевдоэлемента) высотой 4px и длиной 50px и двух аналогичных теней-полосок. Наведение производится на контейнер элемента.

Крестик

.cross

Крестик (кнопка) состоит из двух полосок-псевдоэлементов. Наведение производится на контейнер элемента.

Квадрат

.square

Квадрат (<div> или <span>) часто используется в качестве индикатора чекбокса.

Круг

.circle

Круг (<div> или <span>) часто используется в качестве индикатора радио-кнопки.

Загрузчик

.loader

Загрузчик (<div> или <span>) создан с помощью круга (трехцветного) и анимации.

Изображения

Картинка-миниатюра

.mini-image .overlay

После клика по миниатюре срабатывает js: открываются большое изображение, крестик, оверлей. Закрыть большую картинку можно по клику по оверлею или крестику.

Ретиновые и адаптивные изображения

.retina

Первое изображение вставлено с помощью тега <picture> в html-разметку. Второе является фоном пустого блока. Оба изображения ретиновые и адаптивные (три брейкпоинта: мобильная, планшетная и десктопная версии), внешне ничем не отличаются.

Иконки социальных сетей

.social

Задан общий стиль для всех иконок и отдельное изображение в формате svg для каждой. Для передачи активного состояния указываются другие svg-изображения, в которых изменен цвет.
