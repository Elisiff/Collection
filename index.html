  • Нормализация вьюпорта для windows-телефонов.
  • box-sizing: border-box;
  • Фиксированный скроллбар.
  • Убираем подсветку ссылок и JS-элементов в WebKit на мобильных платформах.
  • Задаем правильное масштабирование текста для IE на мобильных устройствах.
  • Сброс отступов.
  • Стили для <img>.
  • Запрет масштабирования по двойному клику на сенсорных экранах.
  • Стили для IE.
